## BitShiftExplorer
Задачи:
Разработать алгоритм ввода с клавиатуры типов данных unsigned int и long double и показать на экране их внутреннее представление в двоичной системе счисления. 
Написать и отладить программу на языке С++, реализующую разработанный алгоритм.
Дополнить разработанный ранее алгоритм следующими преобразованиями: 
Выполнить циклический сдвиг в заданную пользователем сторону на некоторое количество разрядов в пределах определённой группы разрядов, количество которых и номер младшего разряда в группе задаются с клавиатуры.
